Написать приложение с использованием Table View Controller.
Примеры:
http://www.iphonesdkarticles.com/2009/03/uitableview-drill-down-tableview.
html http://knol.google.com/k/iphone-sdk-using-table-views http://
blog.webscale.co.in/?p=284
Приложение не должно использовать Core Data для загрузки данных.
Данные в таблицу грузятся из обычного массива. Данные выбираются
по личному усмотрению.
Необходимо реализовать редактирование таблицы: добавление,
удаление и перемещение ячеек (элементы массива так же должны
добавляться, удаляться и перемещаться).
